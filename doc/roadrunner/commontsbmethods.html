Common TrajectorySequenceBuilder Methods

This is how you will usually interact with RoadRunner.

Legend:

In the initialization, we need to get a TrajectorySequenceBuilder from our robot. It expects you to pass in a starting pose as an argument. 

    Pose2d startPose = new Pose2d(x:0, y:0, heading:0);
    TrajectorySequence trajseq = robot.drive.trajectorySequenceBuilder(startPose);
